O Que é uma LLM (Large Language Model)?

Conceito
📱

A Analogia do Teclado

Pense no corretor ortográfico ou no autocomplete do seu smartphone que sugere a próxima palavra.

Uma LLM faz o mesmo: porém, ela foi treinada em centenas de bilhões de frases da internet, repositórios de código e livros, prevendo as sequências com um realismo impressionante.

Termos Críticos (Clique para virar)

🔤 Token
Fragmento de texto (~4 caracteres). "desenvolvedor" = ~3 tokens. Faturamento e limites operacionais são baseados em tokens.
💬 Prompt
A instrução enviada à IA. A clareza do prompt dita diretamente a qualidade e acerto da resposta.
⚖️ Parâmetros
Pesos internos aprendidos. Em 2026, entende-se que modelos gigantescos não são sinônimo de eficiência de código.
🪟 Contexto
Memória operacional da IA. Maior contexto (ex: 2M tokens) permite ler e analisar repositórios inteiros de uma vez.
⚡ Níveis & Esforço
Escala de raciocínio. Leve/Low (Haiku, Flash): rápido/barato. Balanced/Medium (Sonnet): padrão do dev. Deep/High (Opus, XHigh): lógica complexa.
🌡️ Temp. & Alucinação
Temp: nível de criatividade (baixa = bom para código). Alucinação: erros falsos gerados com confiança pela IA.

Arquitetura

Agente = LLM + Harness

Estrutura

Clique nas camadas abaixo para ver a função de cada componente na arquitetura agêntica.

🤖 AGENTE DE IA Orquestrador
Responsável por perseguir objetivos complexos de ponta a ponta. Ele recebe uma meta ("Implemente JWT e crie testes"), quebra em sub-tarefas, executa comandos, avalia erros e corrige o plano de forma autônoma.
⚙️ HARNESS (Suporte Agêntico) Camada de Ação
O corpo físico da IA. O Harness fornece loops de execução (Pensar → Agir → Observar), memória persistente, acesso direto ao terminal/arquivos e filtros críticos de segurança.
🔄 Loop 🧠 Memória 🛠️ Ferramentas 🛡️ Segurança 📊 Observabilidade
🧠 LLM (Large Language Model) Motor de Decisão
O cérebro cognitivo. Recebe texto e gera probabilidades de saída baseadas em seu treinamento. Ela calcula a decisão lógica e instrui o Harness sobre o que deve ser feito a seguir.
Equação: Agente = Raciocínio (LLM) + Ação (Harness)

Ferramentas principais

Antigravity (Google DeepMind)

Google SOTA

Plataforma Agêntica Unificada

A ferramenta oficial da Google DeepMind que une motor CLI, app visual e fork da IDE em uma única arquitetura sincronizada.

Modelo Principal: Gemini 3.5 Flash Janela: 2M Tokens

⌨️ CLI (Terminal Go)

Alta performance compilada em Go. Substituiu por completo o Gemini CLI em 18/Jun/2026. Excelente para tarefas assíncronas.

🖥️ APP (Desktop 2.0)

Painel completo para monitoramento multi-agente e tarefas agendadas em background. Integração nativa com ecossistema Firebase.

🛠️ IDE (VS Code Fork)

Editor nativo com painéis integrados (Editor vs. Manager Surface) para codificação supervisionada e escrita assistida.

🔀 Subagentes Dinâmicos: Spawn de processos em paralelo.
📋 Artefatos: Planos estruturados gerados em Markdown interativo.
🔬 Science Skills: Integração nativa com bases AlphaGenome.
🐍 Python SDK: Facilidade para customizar ganchos do agente.
Ferramentas principais

Codex (OpenAI)

OpenAI Flagship

Orquestração Visual de Metas

O Codex da OpenAI atua como um coordenador multi-superfície, permitindo que você defina metas e acompanhe-as graficamente.

Modelo Principal: GPT-5.5 Cotas: Janela Rolante de 5h

⌨️ CLI

Inicia tarefas complexas locais e as exporta dinamicamente para o aplicativo visual para acompanhamento humano do fluxo de trabalho.

🖥️ APP (Desktop & Web)

Central gráfica contendo o painel "Codex Sites" para gerar deploys imediatos de protótipos de desenvolvimento direto da interface.

⚠️
Alerta de Capacidade: O GPT-5.5 apresenta problemas recorrentes de "Model at capacity" em horários de pico, especialmente com níveis de raciocínio de alto esforço (xhigh effort).
Ferramentas principais

Claude Code (Anthropic)

Anthropic CLI-First

Poder Absoluto no Terminal

Um agente focado na CLI que lê arquivos, resolve conflitos git e interage via terminal usando o protocolo universal MCP.

Modelo Principal: Sonnet 4.6 / Opus 4.8 Comando: npm install -g @anthropic-ai/claude-code

🚀 Funcionalidades Chave

📁 CLAUDE.md: Arquivo de regras personalizáveis do repositório.
🔌 MCP Nativo: Conexão padronizada com bases SQL, Slack, etc.
🧭 Planning Mode: Criação prévia do plano antes do código.

⚠️ Mudança Crítica (15/Jun/2026)

O uso automatizado e sem supervisão (headless / CI/CD) foi removido da assinatura Pro. O faturamento desse tráfego agora é debitado via sistema de créditos por token em dólares.
